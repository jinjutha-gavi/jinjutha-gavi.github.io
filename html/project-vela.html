Application Project

Vela Bakery App

A premium hybrid bakery ordering mobile application built with Flutter & Firebase. Features a warm modern aesthetic, real-time interactive delivery tracking on live maps, gamified order selection, and a comprehensive Admin Panel.

Dart

Client-optimized programming language powering 99.6% of the codebase

Cloud Firestore

NoSQL cloud database for real-time menu, orders, and user data sync

Firebase Auth

Secure authentication with role-based access for admin and customer accounts

flutter_map + latlong2

Highlights

Key Features

01

Customer Shopping Experience

Visually appealing menu browsing with real-time Firestore updates, seamless cart & checkout, dynamic shipping address selection, and mock payment gateways.

02

Random Pick Gacha

A fun gamified "Can't Decide?" page with custom CurvedTextPainter and smooth elastic animations to randomly pick a bakery item.

03

Live Map Order Tracking

Interactive OpenStreetMap with animated delivery rider along route segments, expandable DraggableScrollableSheet with ETA and driver info.

04

Dark & Light Theme

Elegant warm mocha dark palette (#1C1714 & #2A2320) powered by Singleton ThemeService with SharedPreferences persistence.

05

Admin Panel

Role-based CRUD access for menu management — add, edit, delete categories & items with cloud image uploads from device camera/gallery.

06

Profile & Settings

Custom profile management, shipping address manager, payment methods, help center FAQ, and comprehensive account settings.
