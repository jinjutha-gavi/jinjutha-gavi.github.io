Cybersecurity Management Project

Vulnerability & Exploitation Lab

A highly structured cybersecurity audit showcasing advanced Tenable Nessus network vulnerability discovery, active system enumeration, and remote Metasploit penetration testing to escalate local machine credentials.

Download PDF Report

Security Platform

Kali Linux OS Environment

Core Toolsets

Tenable Nessus & Metasploit Framework

Target Systems

Hosts scanned: 10.81.165.58 & 10.80.137.219

Access Level

SYSTEM Privilege (NT AUTHORITY\SYSTEM)

Audit Methodology

Security Lab Execution Phases

01

Port Discovery

Deploying targeted active ping loops and deep nmap -sV scans to map active network ports (135, 139, 445, 3389) and extract OS system versions.

02

Nessus Network Audit

Executing credential-less advanced vulnerability audits via **Tenable Nessus Essentials Plus** (`ice_scan`) against target host `10.81.165.58`, discovering and categorizing active severity threats.

03

Vulnerability Verification

Running automated vulnerability scans via Nmap --script vuln scripts to audit active system weaknesses, flagging Slowloris DOS and MS17-010 flaws.

04

Exploitation & Root Shell

Triggering the ms17_010_eternalblue exploitation vector in Metasploit Console to deploy shell payloads, gaining full root system command shell rights.
