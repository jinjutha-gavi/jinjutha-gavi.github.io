IOT Project

Fresh Fit

An IoT-based hardware and mobile application system designed to automatically monitor humidity, dust mites, and temperature inside wardrobes in real-time. It features smart notifications via LINE OA and app integration to prevent mold, odors, and fabric damage.

Pitch Deck

Platform

Hardware (IoT) — Mobile & LINE OA Integration

Timeline

2025

Role

Business & Technology

Architecture

Edge Layer (Hardware), Network Layer (Protocol), Cloud

Technology

Tech Stack

ESP32 & Microcontroller

Microcontroller with integrated Wi-Fi module, handling real-time data processing and seamless cloud connectivity via MQTT protocol

DHT22 & Sharp GP2Y Sensors

Advanced environmental sensing configuration deployed to measure humidity, temperature, and precise dust mite risk indicators inside localized spaces

Cloud & Database Systems

Scalable cloud backend using MQTT brokers and structured database storage to log data histories and trigger push notifications

Highlights

Key Features

01

Environmental Sensing

Real-time monitoring of temperature, relative humidity, and air dust particles to safeguard garments and detect hidden health hazards.

02

Smart Notification Center

Proactive alerts delivered instantly via Mobile App and LINE OA when humidity surpasses customizable safe thresholds.

03

Risk & Behavioral Analytics

Automated 7-day historical trend analysis calculating dust mite proliferation risks and generating actionable clothing care solutions.

04

Smart Calendar Tracking

Seamless automated logging system tracking the exact dates garments are placed in the wardrobe to optimize laundry schedules.

05

Digital Wardrobe Organizer

AI-powered digital management interface classifying clothes by categories (Suits, Skirts, Casuals) for effortless, efficient daily selection.

06

Triple-Color LED Indicator

Instant physical hardware feedback utilizing green, yellow, and red LEDs to reflect the wardrobe's current atmospheric safety level at a glance.
